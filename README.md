> A test wallet project based on Ts.ED

## Getting started

### **Requirements**
- Node >= 14
- Express >= 4 
- TypeScript >= 4.

```batch
# install dependencies
$ yarn install

# serve
$ yarn start

# build for production
$ yarn build
$ yarn start:prod
```
